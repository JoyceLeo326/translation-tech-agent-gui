# 共享资料区

这里存放所有组都可能读取的公共材料。

## 子目录

- `source_materials/`：统一源材料或示例材料。
- `terminology/`：全项目共享术语库。
- `review_sheets/`：人工翻译、审校和回填状态表。
- `references/`：公共参考文献、网页来源、提示词说明。

## 使用原则

- `shared/` 中的文件必须尽量稳定，避免频繁覆盖。
- 如果某组仍在快速迭代，请先放在本组目录，稳定后再复制到 `shared/`。
- 共享 Excel 需要保留清晰字段名，例如 `resource_id`、`group`、`source_type`、`source_text`、`machine_translation`、`human_translation`、`review_status`、`target_path`。
