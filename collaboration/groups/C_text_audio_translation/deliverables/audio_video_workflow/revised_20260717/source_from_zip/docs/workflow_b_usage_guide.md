# 工作流B - 音视频翻译通道 使用说明

## 工作流概述

工作流B是一个音视频翻译通道,支持两种独立运行模式:
- **模式1**: 预处理模式 - 生成待审校Excel
- **模式2**: 回填模式 - 生成成品音视频

该工作流基于LangGraph框架构建,实现了完整的音视频翻译流程,包括ASR语音识别、机器翻译、人工审校、TTS语音合成和音视频混音等功能。

---

## 运行模式详解

### 模式1: 预处理・生成待审校Excel

**适用场景**: 需要对原始音视频进行翻译审校时

**输入参数**:
- **运行模式**: 选择"预处理・生成待审校Excel"
- **原始音视频文件**: 支持mp3/wav/mp4等格式

**处理流程**:
1. **ASR语音识别**: 从音视频中提取文本和时间戳
2. **数据构造**: 组装标准表格数据格式
3. **批量翻译**: 将中文原文翻译为英文(机器翻译)
4. **Excel生成**: 生成包含原文、译文、时间轴参数的标准审校表格

**输出结果**:
- 待审校Excel文件URL
- 提示消息: 要求线下完成翻译审校,填写【最终译文列】

**后续操作**:
用户需要在Excel中完成以下内容:
- 填写【最终译文列】: 确认或修改机器翻译结果
- 可选填写【人工编辑列】、【审校列】、【质量控制列】
- 审校完成后保存Excel文件,用于模式2回填

---

### 模式2: 回填・生成成品音视频

**适用场景**: 已完成Excel审校,需要生成翻译成品时

**输入参数**:
- **运行模式**: 选择"回填・生成成品音视频"
- **定稿Excel文件**: 填写好【最终译文列】的Excel文件

**处理流程**:
1. **Excel读取**: 提取最终译文和时间轴参数
2. **数据校验**: 筛选有效译文数据,过滤空译文条目
3. **批量TTS**: 将英文译文合成为语音音频
4. **音视频混音**: 按时间轴参数替换原视频音轨

**输出结果**:
- 成品音视频文件URL
- 提示消息: 翻译回填成品已生成

## Excel表格规范

### 表头顺序(严格固定)

| 列名 | 说明 | 是否必填 |
|------|------|---------|
| 原材料片段 | 唯一匹配主键,格式: "seg001 \| 00:00:02 - 00:00:05" | 必填(自动生成) |
| 中文原文 | ASR识别的中文句子 | 必填(自动生成) |
| 机器翻译(英译) | 机器翻译的英文译文 | 自动生成(可修改) |
| 人工编辑列 | 人工编辑备注 | 可选 |
| 审校列 | 审校备注 | 可选 |
| 质量控制列 | 质量检查备注 | 可选 |
| **最终译文列** | **最终确认的英文译文** | **必填(人工填写)** |
| 素材基准帧率 | 原视频帧率信息 | 自动生成 |
| 素材基准语速 | 原音频语速信息 | 自动生成 |
| 完整时间轴参数 | 时间戳JSON数据 | 自动生成(勿修改) |

### 原材料片段格式规范

**格式**: `segID | HH:MM:SS - HH:MM:SS`

**示例**:
- `seg001 | 00:00:02 - 00:00:05`
- `seg002 | 00:00:05 - 00:00:12`
- `seg003 | 00:00:12 - 00:00:18`

**作用**: 
- 唯一匹配主键,用于回填时对齐素材和译文
- 包含精准时间戳,用于音视频混音时的时间轴匹配

## 使用流程示例

### 场景1: 翻译一段中文视频

1. **第一步(模式1)**:
   - 上传原始中文视频文件
   - 选择运行模式: "预处理・生成待审校Excel"
   - 运行工作流
   - 获取待审校Excel文件

2. **第二步(人工审校)**:
   - 打开Excel文件
   - 检查【机器翻译(英译)】列
   - 在【最终译文列】填写确认的英文译文
   - 保存Excel文件

3. **第三步(模式2)**:
   - 上传审校后的Excel文件
   - 选择运行模式: "回填・生成成品音视频"
   - 运行工作流
   - 获取翻译后的英文配音视频

## 注意事项

### 重要约束

1. **Excel列顺序不可更改**: 必须严格按照表头顺序,否则数据解析失败
2. **原材料片段格式固定**: 必须保持"segID | 时间戳"格式,用于时间轴匹配
3. **最终译文列必填**: 模式2运行前必须填写【最终译文列】,否则跳过该条目
4. **双分支独立运行**: 两种模式互不干扰,不会自动串联

### 文件格式要求

- **音视频输入**: mp3/wav/mp4等常见格式
- **Excel输入**: .xlsx格式(不支持.xls)
- **输出格式**: Excel为.xlsx, 音视频为mp4/mp3

### 时间轴参数说明

【完整时间轴参数】列包含JSON格式的时间戳数据:
```json
{
  "segment_id": "seg001",
  "start_time_ms": 2000,
  "end_time_ms": 5000,
  "duration_ms": 3000
}
```

**请勿修改此列内容**,否则会导致音视频混音时间轴错位。

## 故障排查

### 常见问题

| 问题 | 可能原因 | 解决方案 |
|------|---------|---------|
| ASR识别为空 | 音视频文件损坏或格式不支持 | 检查文件完整性,尝试转换格式 |
| Excel读取失败 | 文件格式错误或列名不符 | 确认为.xlsx格式,检查列顺序 |
| 翻译失败 | LLM服务异常 | 重试或检查配置文件 |
| TTS合成失败 | 文本过长或服务异常 | 检查译文长度,重试 |
| 音视频混音失败 | 时间轴参数解析失败 | 检查Excel中时间轴参数列 |

## 配置文件说明

### LLM翻译配置

位置: `config/translation_llm_cfg.json`

配置内容:
- **模型**: doubao-seed-2-0-lite-260215 (均衡型模型)
- **温度**: 0.3 (确保翻译稳定性)
- **最大tokens**: 512 (适合短语翻译)
- **系统提示词**: 专业翻译专家角色定义
- **用户提示词**: 包含中文原文和片段ID的模板
