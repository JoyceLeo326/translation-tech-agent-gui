# 协作区总览

本目录是 ABC 组共享协作区，也是后续 GUI 和智能体整合的固定输入来源。

## 目录职责

- `groups/`：各组自己的交付物、说明、源码压缩包和阶段成果。
- `shared/`：所有组都可以读取的公共资料，例如术语库、参考资料、人工审校 Excel。
- `integration/`：总整合区域，用于汇总清单、适配器、最终输出和打包前检查。

## 放置规则

1. 各组只向自己的 `groups/<group>/deliverables/` 提交稳定成果。
2. 需要其它组读取的材料，复制一份到 `shared/`，不要让其它组依赖个人临时目录。
3. 不提交 `.env`、API Key、账号密码、私有 Token。
4. 大文件优先提交压缩包和必要说明；中间缓存、运行输出放 `scratch/` 或 `runtime_outputs/`，这些目录默认不入库。
5. 每次新增交付物后，同步更新 `integration/manifests/group_deliveries.md`。
