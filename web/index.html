COZE MULTI-MODEL WORKFLOW

扣子工作流不是装饰，
而是一套可检查的精译流程

先识别文化术语和目标文体，再让 Kimi、DeepSeek、豆包分别初译与互评， 最后由 GLM 融合终稿。离线可看完整流程证据，配置安全令牌后可运行线上工作流。

18 个节点 28 条连接 3 路独立初译

    术语提取

    从统一术语库读取文化概念的推荐译法。

  2. 02
    风格判断

    识别儿童文学读者、语气、节奏与朗读要求。

  3. 03
    多路初译与互评

    降低单一模型偏差，比较准确性与自然度。

  4. 04
    融合终稿

    输出可进入人工审校表的英文译文。
