# 译述 YISHU v1.2.0｜中国文化多模态外译工作台

“译述”把图像、文档、文化术语和声音放进同一条可审校的外译流程。解压后双击 `Yishu.exe`，即可从素材导入开始完成提取、翻译、人工确认与成品导出。

## 本版重点

- 建立“译述 YISHU”品牌系统，以中文角引号和跨语言路径组成独立应用图标。
- 使用深玉、象牙白、朱砂、瓷蓝与旧金构成克制的编辑式视觉系统。
- 将横向网页式导航重构为桌面工作台侧栏，页面标题、任务与操作保持清晰分层。
- 首屏采用“译文字，也译语境”的产品表达，同时保留图片、Word、音视频和文字四个直接任务入口。
- 统一接入全部图文、文化术语与风格、DOCX、音视频交付内容，不在最终用户界面区分项目组。
- 多模型精译继续展示真实扣子工作流结构、模型分工、离线演示与线上运行入口。

## 验收结果

- 交付完整性、单元测试、工作流测试和界面测试全部通过。
- 5/5 套 DOCX 样例完成真实回填，失败 XML 为 0。
- 英文配音、朗读文本和二维码可由本机流程重新生成。
- 发布包完成独立短路径解压、密钥扫描、EXE 自检、生产自检与窗口启动检查。
