# 译述 YISHU v1.4.0｜在线模型接口与 Coze 核心精译

本版将桌面程序从“离线可演示”扩展为“离线可交付、接入自己的 API 后可真实在线处理”的完整工作台，并同步重构 Vercel 产品导览。

## 主要更新

- 左下角新增“模型接口”，支持 OpenAI、Ollama、LM Studio 和其他 OpenAI 兼容服务。
- 支持 Responses API 与 Chat Completions 两种文本协议，本机服务允许无 Key 使用。
- 文字、图片、Word 审校表、音频转写与英文配音接入统一在线模型配置。
- 在线 TTS 不可用时自动回退 Windows 本机语音，并在结果中记录回退原因。
- Coze 多模型精译调整为三种翻译模式的中心亮点，统一展示 18 节点、28 连接和模型分工。
- 首页重新分配大屏空间，增加离线、通用模型 API 与 Coze 三种运行方式说明。
- 全局页头、分区标题、说明文字与数据字号重新校准。
- 新增智能体架构与工作流架构文档。
- Vercel 页面增加交互玩法导览、模型接口展示、Coze 核心章节和 2560 x 1600 高清截图。

## 安全说明

API Key 与 Coze Token 只保存在 Windows 本机 `.env` 中，该文件不会进入 GitHub、Vercel 或发布包。程序日志和状态栏不显示完整密钥。
