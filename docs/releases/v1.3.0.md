# 译述 YISHU v1.3.0｜编辑工作室界面重构

本版重新设计桌面端的信息架构与品牌系统，不是在上一版上继续叠加配色。目标是让第一次打开软件的用户可以直接判断该放入什么素材、系统会怎样处理，以及最后会得到哪些成品。

## 本版重点

- 将 224 像素深色后台侧栏重构为 112 像素象牙白工具轨，释放内容空间并保留清晰中文标签。
- 首页改为“新建翻译、真实成品示例、直接任务入口、验收数据”四层结构，移除模板式宣传区和浅色卡片堆叠。
- 使用真实 17 页译成品作为首屏预览，不再用抽象装饰图代替项目内容。
- UI 字体与展示字体分离：操作界面使用 Noto Sans SC，品牌标题和关键数据使用 Noto Serif SC。
- 批量处理页新增任务目标、接入范围和交付预览，用户运行前即可理解流程会检查什么、生成什么。
- 导出文件页新增 Word、表格、音频和验收记录的真实数量概览。
- 品牌图标重做为“译字印记”，同步更新 PNG、ICO 与 README 品牌横幅。
- 新增可重复执行的七页面截图脚本和品牌资产生成脚本。

## 验收结果

- 1440 × 900 标准窗口和 1100 × 720 最小窗口均完成真实截图审稿，无横向滚动和文字遮挡。
- 22 项自动化测试与 25 项交付检查全部通过，DOCX 回填 5/5，失败 XML 为 0。
- 独立短路径解压后，三项 EXE 自检、真实窗口启动、`.env` 检查和令牌扫描全部通过。
