# 文澜 v1.0.0｜中国文化多模态外译工作台

这是面向最终使用者的首个完整 Windows 版本。解压后双击 `Wenlan.exe`，即可在同一个界面处理图文、普通文本、Word 文档和音视频素材，不需要了解项目原有的协作分组。

## 第一次使用

1. 下载并解压附件，建议使用 `D:\CultureWorkbench` 等较短目录。
2. 双击 `Wenlan.exe`。
3. 把素材拖到“开始”页，或在“文件翻译”选择图片、Word 文档或音视频。
4. 按界面中的“选择素材、自动处理、人工确认、生成成品”依次完成任务。
5. 在“成品中心”查看文档、表格、音频、二维码和验收报告。

程序内置“打开示例任务”，首次打开即可查看完整示例和已验收成品。
“作品展示”页集中呈现真实翻译页面、核心成果和现场可点击的验证文件，适合比赛演示。

## 本版包含

- 71 条图文审校记录、17 页成品预览和相关媒体资源。
- 251 条去重文化术语，其中 40 条补充语料附官方来源链接。
- DOCX 正文、表格、页眉和页脚提取，Excel 人工审校，以及保留版式的 XML 级技术回填。
- 音视频逐句审校、智能翻译、Windows 本地英文语音合成、朗读文本和二维码输出。
- OpenAI 单次翻译、三步翻译工作流及扣子工作流入口。
- 完整原始交付、统一成品区、Excel/JSON 索引与 SHA256 验收信息。

## 在线能力

文档提取、DOCX 回填、资源浏览和 Windows 本地语音合成无需 API。智能翻译、语音识别及扣子工作流需要使用者在本机配置相应密钥；未配置时程序会明确提示，不会伪造模型结果。
